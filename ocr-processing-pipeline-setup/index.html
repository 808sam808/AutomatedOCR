2

Launch with Docker Compose

This starts Ollama, the OCR watcher, and auto-pulls the OCR model.

ğŸ’¡ GPU users: Uncomment the GPU section in docker-compose.yml for faster OCR.

Copy or move image files into the watch folder. The pipeline triggers automatically.

View Results

Open the web dashboard from any device on your network.
